# v1.1.0

**规划功能：**

# v1.0.6-alpha.5

1. 新增图表类型支持：桑基图、雷达图、玫瑰图
2. 修复依赖报错
3. 支持node端调用
4. 支持初始化时传入url和请求参数，自定义模型请求方式：可以自定义包括大模型服务URL、请求头、请求方法、模型名称、最大tokens和温度等参数
5. 移除ffmpeg依赖，改为外部传入，减少了包体积和依赖安装难度
